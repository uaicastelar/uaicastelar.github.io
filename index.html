Proyectos

  • Este proyecto tiene como principal objetivo realizar un análisis de tecnologías existentes, lo cual permitirá comparar la performance de los distintos tipos de entornos de desarrollo y además desarrollar APIs que contribuyan a la implementación de estándares existentes.

    Líneas de investigación:
    - Análisis de estándares del W3C, evaluación de aplicación y uso
    - Desarrollo de APIs para la implementación de estándares
    - Entornos de Desarrollo – Comparativa entre entornos web, nativos e híbrido

  • Este proyecto busca diseñar interfaces que mediante interacción física con el dispositivo permitan controlar distintos tipos de aplicaciones.

    Líneas de investigación:
    - Eventos desencadenados mediante el contacto con una superficie utilizando NFC
    - Captura de eventos iniciados por sensores de movimiento de los dispositivos y diseño de interfaces reactivas.
    - Geofencing
    - Monitoreo de adultos mayores
